# Example Sidecar Plugin

This example demonstrates how to create a sidecar plugin that runs an external process.

## Overview

Sidecar plugins allow running external binaries alongside the main Tauri application.

## Use Cases

- Local LLM inference (llama.cpp)
- Speech-to-text (Whisper.cpp)
- Video processing
- Database servers
- Custom tools
